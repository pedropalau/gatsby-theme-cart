## Install

```bash

```

```js

plugins: [
  {
    resolve: `gatsby-theme-cart`,
    options: {
      // ...
    },
  },
]
```

### Options

| Option           | Default | Required | Description                                                                       |
| ---------------- | ------- | -------- | --------------------------------------------------------------------------------- |
| `enableCartPage` | `true`  | No       | Useful if you wish to use the exported `Cart />` component in your existing page. |
